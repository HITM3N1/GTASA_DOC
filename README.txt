Dokumentacja symulacji agentowej.

Aby uruchomić stronę, rozpakuj archiwum ZIP i otwórz plik:
dokumentacja_symulacji_site_v4/index.html

Wersja v4 zawiera:
- sekcję 5. Diagramy z trzema podstronami opartymi na obrazach PNG,
- Diagram klas,
- Diagram obiektów,
- Diagram maszyny stanów,
- podstronę Autorzy w sekcji Informacje ogólne.

Nagłówki głównych sekcji w menu są celowo nieklikalne. Klikalne są wyłącznie podstrony.

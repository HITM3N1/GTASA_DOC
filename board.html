Dokumentacja elementu programu

Board

klasa

Klasa odpowiadająca za mapę symulacji: tworzenie siatki, chodników, budynków, wejść oraz wyszukiwanie komórek.

Założenie i logika działania

Mapa jest budowana etapami: pusta siatka, chodniki, budynki, sąsiedzi i wejścia.

Pola klasy

PoleOpis
private readonly int sizeRozmiar mapy.
private Dictionary<Vector2, Cell> gridSiatka mapy przechowywana jako słownik pozycji i komórek.
private List<Pavment> pavmentsLista wszystkich chodników.
private List<Building> buildingsLista wszystkich budynków.
private Citizens citizensReferencja do grupy obywateli.

Metody klasy

public Board(int size)

Konstruktor zapisuje rozmiar mapy oraz tworzy puste kolekcje.

Argumenty: int size. Typ zwracany: konstruktor nie zwraca wartości.

public void Initialize(Citizens citizens)

Tworzy pełną mapę: puste komórki, chodniki, budynki, sąsiadów i wejścia.

Argumenty: Citizens citizens. Typ zwracany: void.

public void Update(GameTime gameTime)

Aktualizuje wszystkie budynki.

Argumenty: GameTime gameTime. Typ zwracany: void.

public void Draw(SpriteBatch spriteBatch)

Rysuje chodniki i budynki.

Argumenty: SpriteBatch spriteBatch. Typ zwracany: void.

private void GeneratePavment(int count, int minimaloffset)

Generuje chodniki poziome i pionowe.

Argumenty: int count, int minimaloffset. Typ zwracany: void.

void GenerateBuilding()

Tworzy budynki z pustych obszarów mapy.

Argumenty: brak. Typ zwracany: void.

void GenerateNeighbors()

Tworzy sąsiedztwa chodników i wejścia do budynków.

Argumenty: brak. Typ zwracany: void.

public void CheckNeighbor(Pavment pavment, Cell neighbor, int direction)

Sprawdza sąsiednią komórkę i wykonuje odpowiednie połączenie.

Argumenty: Pavment pavment, Cell neighbor, int direction. Typ zwracany: void.

public void TryMakeEntrace(Building building, Pavment pavment, int directionFromPavmentToBuilding)

Ustawia wejście do budynku.

Argumenty: Building building, Pavment pavment, int directionFromPavmentToBuilding. Typ zwracany: void.

public Cell GetCell(Vector2 absolutPosition)

Zamienia pozycję pikselową na kafelkową i zwraca komórkę.

Argumenty: Vector2 absolutPosition. Typ zwracany: Cell.

public List<Building> GetGangBuildings(GroupAbstract group)

Zwraca budynki należące do podanej grupy.

Argumenty: GroupAbstract group. Typ zwracany: List<Building>.

public List<Building> GetAllBuildingsUnderAttack()

Zwraca budynki pod atakiem.

Argumenty: brak. Typ zwracany: List<Building>.

public Cell GetRandomPavment()

Zwraca losowy chodnik.

Argumenty: brak. Typ zwracany: Cell.

public Cell GetRandomBorderPavment()

Zwraca losowy chodnik końcowy.

Argumenty: brak. Typ zwracany: Cell.

public Building GetFreeSpawnBuilding(Group group)

Zwraca wolny budynek startowy i przypisuje go do grupy.

Argumenty: Group group. Typ zwracany: Building.

public Building GetRandomFreeBuilding(GroupAbstract groupAbstract)

Zwraca losowy budynek, który nie należy do podanej grupy.

Argumenty: GroupAbstract groupAbstract. Typ zwracany: Building.

public void SetCitizens(Citizens citizens)

Ustawia referencję do obywateli.

Argumenty: Citizens citizens. Typ zwracany: void.

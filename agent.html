Dokumentacja elementu programu

Agent

klasa

Klasa reprezentująca pojedynczego agenta działającego w symulacji.

Założenie i logika działania

Agent jest podstawową jednostką symulacji. Może poruszać się po chodnikach, wejść do budynku, zostać zrekrutowany i działać jako członek konkretnej grupy.

Metody klasy

public Agent(GroupAbstract group, Board board, Cell spawnCell)

Konstruktor ustawia grupę, planszę, komórkę startową, pozycję, ścieżkę, prędkość oraz stan logiczny.

Argumenty: GroupAbstract group, Board board, Cell spawnCell. Typ zwracany: konstruktor nie zwraca wartości.

public void Update(GameTime gameTime)

Aktualizuje rekrutację, ruch, ścieżkę i zmianę komórki.

Argumenty: GameTime gameTime. Typ zwracany: void.

public void Draw(SpriteBatch spriteBatch)

Rysuje agenta teksturą zgodną z kolorem jego grupy.

Argumenty: SpriteBatch spriteBatch. Typ zwracany: void.

public GroupAbstract GetGroup()

Zwraca grupę agenta.

Argumenty: brak. Typ zwracany: GroupAbstract.

public bool CanBeMoved()

Zwraca informację, czy agent może zostać skierowany do celu.

Argumenty: brak. Typ zwracany: bool.

public void SetTargetPath(Queue<Vector2> targetPath)

Ustawia nową ścieżkę ruchu agenta.

Argumenty: Queue<Vector2> targetPath. Typ zwracany: void.

public void LockAgent(GroupAbstract occup)

Blokuje agenta na czas rekrutacji i ustawia jego przyszłą grupę.

Argumenty: GroupAbstract occup. Typ zwracany: void.

public bool IsBeingRecruted()

Zwraca informację, czy agent jest w trakcie rekrutacji.

Argumenty: brak. Typ zwracany: bool.

public void UnlockAgent()

Odblokowuje agenta po rekrutacji.

Argumenty: brak. Typ zwracany: void.

public void TryRecrute(Pavment actuallCell)

Sprawdza agentów w pobliżu i próbuje rozpocząć rekrutację obywatela.

Argumenty: Pavment actuallCell. Typ zwracany: void.

public void Wander()

Obsługuje domyślne poruszanie agenta.

Argumenty: brak. Typ zwracany: void.

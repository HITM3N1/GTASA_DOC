Dokumentacja elementu programu

Building

klasa

Klasa reprezentująca budynek, który może mieć właściciela, wejście, agentów wewnątrz i stan ataku.

Założenie i logika działania

Budynek przechowuje właściciela i stan ataku. Po wejściu agentów i upływie określonego czasu może zmienić właściciela.

PoleOpis

Metody klasy

public Building(Rectangle bounds, GroupAbstract occupation)

Konstruktor ustawia obszar budynku, właściciela i wartości początkowe.

Argumenty: Rectangle bounds, GroupAbstract occupation. Typ zwracany: konstruktor nie zwraca wartości.

public void CallAgents(int k, Group group)

Przywołuje agentów podanej grupy do wejścia budynku.

Argumenty: int k, Group group. Typ zwracany: void.

public void Update(GameTime gameTime)

Obsługuje proces ataku, liczenie agentów i zmianę właściciela.

Argumenty: GameTime gameTime. Typ zwracany: void.

public void Draw(SpriteBatch spriteBatch)

Rysuje budynek z tekstur środka, ścian, rogów i drzwi.

Argumenty: SpriteBatch spriteBatch. Typ zwracany: void.

public bool IsUnderAttack()

Zwraca informację, czy budynek jest atakowany.

Argumenty: brak. Typ zwracany: bool.

public GroupAbstract GetOccupation()

Zwraca właściciela budynku.

Argumenty: brak. Typ zwracany: GroupAbstract.

public bool IsExitSet()

Zwraca informację, czy budynek ma wyjście.

Argumenty: brak. Typ zwracany: bool.

public void SetExit(int directionFromPavmentToBuilding, Pavment pavment)

Ustawia wyjście budynku i pozycję drzwi.

Argumenty: int directionFromPavmentToBuilding, Pavment pavment. Typ zwracany: void.

public CellType GetCellType()

Zwraca typ komórki.

Argumenty: brak. Typ zwracany: CellType.

public bool Isfree()

Sprawdza, czy budynek jest traktowany jako wolny.

Argumenty: brak. Typ zwracany: bool.

public void SetOccupation(Group group)

Ustawia nowego właściciela budynku.

Argumenty: Group group. Typ zwracany: void.

public Vector2 GetSpawnAbsolutePosition()

Zwraca pozycję startową agenta wewnątrz budynku.

Argumenty: brak. Typ zwracany: Vector2.

public Vector2 GetAbsolutPosition()

Zwraca pozycję budynku.

Argumenty: brak. Typ zwracany: Vector2.

public Vector2 GetExitPos1()

Zwraca pozycję drzwi.

Argumenty: brak. Typ zwracany: Vector2.

public Vector2 GetExitPos2()

Zwraca pozycję chodnika przy drzwiach.

Argumenty: brak. Typ zwracany: Vector2.

public Queue<Vector2> GetEntryPath()

Zwraca ścieżkę wejścia do budynku.

Argumenty: brak. Typ zwracany: Queue<Vector2>.

public void AddAgent(Agent agent)

Dodaje agenta do budynku.

Argumenty: Agent agent. Typ zwracany: void.

public void RemoveAgent(Agent agent)

Usuwa agenta z budynku.

Argumenty: Agent agent. Typ zwracany: void.

public void StartAttack()

Rozpoczyna atak na budynek.

Argumenty: brak. Typ zwracany: void.

public bool IsPoliceInside()

Zwraca informację, czy w budynku jest policja.

Argumenty: brak. Typ zwracany: bool.

public bool IsAttackEnded()

Zwraca informację, czy atak jest zakończony.

Argumenty: brak. Typ zwracany: bool.

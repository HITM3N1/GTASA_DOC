Dokumentacja elementu programu

Group

klasa

Klasa bazowa przechowująca wspólną logikę dla wszystkich grup agentów.

Założenie i logika działania

Klasa udostępnia wspólne mechanizmy dla grup: kolor, listę agentów, rysowanie oraz dodawanie i usuwanie agentów.

Pola klasy

PoleOpis
protected GroupColor colorKolor grupy.
protected List<Agent> agentsLista agentów należących do grupy.
protected Board boardPlansza, na której działa grupa.
protected float speedModifierModyfikator prędkości agentów grupy.

Metody klasy

protected Group(GroupColor color, Board board, float speedModifier)

Konstruktor tworzy listę agentów i zapisuje podstawowe parametry grupy.

Argumenty: GroupColor color, Board board, float speedModifier. Typ zwracany: konstruktor nie zwraca wartości.

public void Draw(SpriteBatch spriteBatch)

Rysuje wszystkich agentów grupy.

Argumenty: SpriteBatch spriteBatch. Typ zwracany: void.

public GroupColor GetColor()

Zwraca kolor grupy.

Argumenty: brak. Typ zwracany: GroupColor.

public int GetAgentCount()

Zwraca liczbę agentów.

Argumenty: brak. Typ zwracany: int.

public float GetSpeedModifier()

Zwraca modyfikator prędkości.

Argumenty: brak. Typ zwracany: float.

public void RemoveAgent(Agent agent)

Usuwa agenta z listy grupy.

Argumenty: Agent agent. Typ zwracany: void.

public void AddAgent(Agent agent)

Dodaje agenta do listy grupy.

Argumenty: Agent agent. Typ zwracany: void.

public List<Agent> GetAgents()

Zwraca listę agentów grupy.

Argumenty: brak. Typ zwracany: List<Agent>.

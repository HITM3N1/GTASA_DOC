Dokumentacja elementu programu

Pavment

klasa

Klasa reprezentująca chodnik, czyli podstawowe pole ruchu agentów.

Założenie i logika działania

Chodniki tworzą sieć komunikacyjną miasta. Agenci poruszają się po tej sieci i dzięki niej mogą być przywoływani do celów.

Metody klasy

public Pavment(Vector2 cords)

Konstruktor ustawia współrzędne, typ, listę agentów, słownik sąsiadów i losuje teksturę.

Argumenty: Vector2 cords. Typ zwracany: konstruktor nie zwraca wartości.

public void Draw(SpriteBatch spriteBatch)

Rysuje teksturę chodnika.

Argumenty: SpriteBatch spriteBatch. Typ zwracany: void.

public List<Agent> GetAgents()

Zwraca listę agentów znajdujących się na chodniku.

Argumenty: brak. Typ zwracany: List<Agent>.

public void AddAgent(Agent agent)

Dodaje agenta do chodnika.

Argumenty: Agent agent. Typ zwracany: void.

public void RemoveAgent(Agent agent)

Usuwa agenta z chodnika.

Argumenty: Agent agent. Typ zwracany: void.

public Vector2 GetAbsolutPosition()

Zwraca pozycję chodnika w pikselach.

Argumenty: brak. Typ zwracany: Vector2.

public CellType GetCellType()

Zwraca typ komórki.

Argumenty: brak. Typ zwracany: CellType.

public bool IsBuildingEntrace()

Zwraca informację, czy chodnik jest wejściem do budynku.

Argumenty: brak. Typ zwracany: bool.

public void SetNeighbor(int direction, Pavment cell)

Dodaje sąsiada w danym kierunku.

Argumenty: int direction, Pavment cell. Typ zwracany: void.

public void CallAgents(int k, Group group, Queue<Vector2> pathToTarget, HashSet<Pavment> visited)

Przeszukuje chodniki w zasięgu i ustawia ścieżkę agentom wybranej grupy.

Argumenty: int k, Group group, Queue<Vector2> pathToTarget, HashSet<Pavment> visited. Typ zwracany: void.
